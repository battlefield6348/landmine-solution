# Landmine Solution (踩地雷解題小工具)

這是一個用來輔助計算正方形踩地雷盤面機率的小工具。

## 專案架構

本專案採用標準 Golang 目錄結構：

- `cmd/solver/`: 程式入口點，負責處理 CLI 輸入與輸出。
- `cmd/wasm/`: WebAssembly 入口點，負責與 JavaScript 互動。
- `internal/model/`: 定義盤面 (Grid) 與儲存格 (Cell) 的資料結構。
- `internal/solver/`: 核心演算法，計算未解格子的地雷機率。
- `docs/`: 網頁版靜態檔案，包含編譯後的 WASM 與 GitHub Pages 頁面。

## 核心邏輯

1. **模型定義**：
   - 儲存格狀態：數字 (0-8)、未解 (Unknown)、旗幟 (Flag)。
   - 盤面尺寸：預設為 3x3，可動態擴展為 4x4, 5x5... 等正方形尺寸。
2. **機率計算法**：
   - 採用 **枚舉法 (Exhaustive Search)**：
     - 找出所有未知格子的集合。
     - 遍歷所有可能的地雷組合 ($2^n$)。
     - 對於每一種組合，驗證其是否符合盤面上所有的數字約束。
     - 統計所有「合法組合」中，每個未知格子出現地雷的次數。
     - 機率 = (該格有雷的合法組合數) / (總合法組合數)。
   - **注意**：當 N 較大且未知格 (u) 數量過多時，計算量會呈爆炸性成長 ($2^n$)，目前針對小範圍局部解題最佳。

## 使用說明

- 執行 `go run cmd/solver/main.go`。
- 依照提示輸入 3x3 的內容。
- 支援輸入符號：
  - `0-8`: 對應數字（0 為空白）
  - `e`: 空白 (Empty/0)
  - `u`: 未解 (Unknown)
  - `f`: 旗幟 (Flag)
